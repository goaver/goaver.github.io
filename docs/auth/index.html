Authentication

This resource handles API authentication

#GET /api/auth/token

Creates / retrieves a bearer token to be used for authentication with all other API endpoints.

#Request Parameters

  • [Header] Basic Authorization (required) - Set the Authorization header to the base64 encoded key:secret pair as per https://en.wikipedia.org/wiki/Basic_access_authentication

#Example Request Header

content-type: application/json
authorization: Basic Z3E1ZTkwMDMtZTQ2Yy00ZjMzLTk0ZGUtOWIxZWVkZmViOTRiOk9EUTJOVGsyTmpOallUbG1NakV6TW1OaU5ERTBZV0kzWVRaaU9HTXhaR014TkdZeFlqYzRORFkxT1RZPQ==

#Response Parameters

  • token - The bearer token to be used for all other API calls
