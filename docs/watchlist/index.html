Watchlist Search

This resource will allow the creation and retrieval of watchlist searches

#POST api/watchlist/search

#Request Parameters

  • groupId (required) - The check group the search is in
  • firstName (optional *) - The individual's first name
  • middleName (optional ) - The individual's middle name
  • lastName (optional *) - The individual's last name
  • businessName (optional *) - The business / organization name to search
  • country (optional) - The country for the individual or business
  • stateOrProvince (optional) - The state for the individual or business
  • fileContent (optional) - Base64 image (JPG or PNG) Data URL of the photo containing the individual's face. When provided, this will perform Visual Watchlist search in addition to the Text Watchlist search (associated fees apply). Information about Data URL can be found [here](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs)
  • fileName (required if fileContent is provided) - The filename for the file content provided for the Visual Watchlist search
  • categories (optional) The watchlist categories to be searched, if omitted the default is to search based on the group settings.

*NOTE: Either firstName + lastName or businessName must be provided at a minimum.

#Example Request

{
  "groupId":"2d1162b5-d6a8-4936-be84-39ec873b7a60",
  "firstName":"Some",
  "middleName":"Random",
  "lastName":"User",
  "businessName":"They Work Here Corp",
  "country":"USA",
  "stateOrProvince":"CA",
  "categories":[
        "Criminal",
        "Terrorism",
        "Financial",
        "FinancialNotices",
        "BorrowerDefaults",
        "Political",
        "Medical",
        "Sexual",
        "Gambling",
        "Education"
  ]
}

#Response Parameters

  • id - The unique watchlist search id that was created

#Example Response

{
  "id": "f4d8f910-8a7b-42f0-ba40-2b2c17cb9118"
}

#GET api/watchlist/{id}

Retrieves the search and status information for the specified watchlist search

#Request Parameters

  • [Path] id (required) - The watchlist search id returned from the search call

#Response Parameters

  • id - The unique id of the search
  • groupId - The group of the search
  • organizationId - The organization of the search
  • checkId - The associated check that created the watchlist search (if applicable)
  • status - The status of the search
  • searchDate - The UTC date / time of the search
  • searchType - The type of search (text, visual, text and visual)
  • recheckInterval - The auto-recurring search interval for this search to be performed (days)
  • lastRecheckDate - The UTC date / time of the last recurring search that was done based on this search
  • searchCriteria - The search criteria provided for the search

#Example Response

{
"id": "f4d8f910-8a7b-42f0-ba40-2b2c17cb9118",
"groupId": "2d1162b5-d6a8-4936-be84-39ec873b7a60",
"organizationId": "afa22173-6a46-4761-8308-27ad4b211c40",
"checkId": null,
"status": "Completed",
"searchDate": "10/4/2019 5:26:11 PM",
"searchType": "Text",
"recheckInterval": 0,
"lastRecheckDate": "",
"searchCriteria": {
    "firstName": "Some",
    "middleName": "Random",
    "lastName": "User",
    "businessName": "They Work Here Corp",
    "country": "USA",
    "stateProvince": "CA"
}
}

#GET api/watchlist/{id}/results

Retrieves the results for a specified watchlist

  • [Path] id (required) - The watchlist search id returned from the search call

#Response Parameters

  • id - The unique id of the search
  • groupId - The group of the search
  • organizationId - The organization of the search
  • checkId - The associated check that created the watchlist search (if applicable)
  • status - The status of the search
  • searchDate - The UTC date / time of the search
  • searchType - The type of search (text, text and visual)
  • recheckInterval - The auto-recurring search interval for this search to be performed (days)
  • lastRecheckDate - The UTC date / time of the last recurring search that was done based on this search
  • searchCriteria - The search criteria provided for the search

#GET api/watchlist/{id}/searchedlists

Retrieves the list of watchlists names and versions that were searched for the results

#Request Parameters

  • [Path] id (required) - The watchlist search id returned from the search call

#Response Parameters

  • name - The name of the list that was searched
  • version - The version of that watchlist that was searched

#Example Response

[
  {
    "name": "ATF Most Wanted List",
    "version": "9/5/2019"
  },
  {
    "name": "Air Force Fugitive List",
    "version": "9/5/2019"
  }...
]
